# React Arcade: Multi-Game Hub

A fully interactive, browser-based gaming arcade built with **React**. This project features multiple classic game styles, persistent high scores, and responsive gameplay mechanics.

---

## Featured Games

### 1. Hero Quest (Zelda-Style RPG)
An top-down action adventure where you play as a customizable hero.
* **Mechanics:** Directional sword swinging, enemy AI pathfinding, and treasure collection.
* **Character Select:** Choose between Primary colors (Red, Blue, Yellow).
* **Scoring:** 100 points per enemy defeat.
* **Controls:** Arrow keys to move, Space to swing sword.

### 2. Cookie Clicker (Infinite Idle)
A high-intensity clicking game with a strategic shop system.
* **Mechanics:** Manual clicking and passive income (Auto-clickers).
* **The Shop:** Purchase "Cursor Pointers" that click for you. Costs scale as you grow.
* **Persistence:** Your total clicks and upgrades are saved.
* **Controls:** Mouse click to earn.

---

## Built With
* **React.js** - Component-based UI architecture.
* **JavaScript (ES6+)** - Game logic and physics.
* **CSS3** - Keyframe animations for sword swings and UI transitions.
* **LocalStorage API** - For persistent High Scores.
